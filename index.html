Data Analysis Project for Predicting GPA and PC Possession

Key Words: SAS; GPA predict; PC possession rate; Logistic model

  • Used a linear model to fit the GPA prediction model, we applied two common methods to select variables, one was to select them according to the p-value (t-test), the other one was the stepwise selection method.

Survey of Sleeping Quality for SUSTech Students

Key Words: sampling survey; stratified sampling; ANOVA test; comparison
